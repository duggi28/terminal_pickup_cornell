# terminal_pickup_cornell_final

This blog application allows students of Cornell University to schedule pick-up sports games on and around campus. The app was developed in OCaml and is meant to be run on terminal. Existing  features include logging in/out, sign-up caps, liking posts, following profiles, filtering/sorting, and commenting<img width="468" height="11" alt="image" src="https://github.com/user-attachments/assets/0853a5ad-2ed1-4cb7-9656-2c9b36fe85ee" />. A user interface and database can be included for use in a larger market.
